Аддон: SumakMinimap
Версия: 1.3.R003
Назначение: замена стандартной близзардовской миникарты.
Отображает: время, координаты, наименование локации (цветом), значки BG, LFG, сложность инста,
трекер, наличие почты.
Версия клиента: [4.0.6]
Управление:
Добавляет команды для клиента:
/кд, /rl -перезагрузка интерфейса;
/resetminimap - сброс пользовательского местоположения карты на стандартный.
При нажатой клавише ALT возможно изменение местоположения миникарты.
Замечание: аддон конфликтует с аддоном MBF (MinimapButtonFrame)
Установка: Распаковать архив в папку c установленным клиентом\Interface\AddOns\
Портебление памяти: 14кб

Расшифровка версий: N.YY.RXX
N - номер версии аддона. Меняется при глобальных изменениях.
YY - номер подверсии. Меняется при изменениях, затрагивающих функциональность аддона.
RXX - номер ревизии. Меняется при изменениях, не затрагивающих функциональность аддона. 
При изменнии RXX аддон можно не обновлять.
